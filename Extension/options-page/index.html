Dimming untagged lists

When you add a high priority (e.g. Doing) or a normal priority (e.g. Todo) list to a board, List Highlighter can automatically dim untagged lists in order to create better contrast. You can change both settings below.

Dim untagged lists on boards with normal priority lists (recommended)

Default: on

Dim untagged lists on boards with high priority lists

Default: off

List limits and card counting

Limiting the number of cards a list can contain can help you plan your workload and focus on current tasks.

You can add a limit to a list by putting a number in square brackets in the list title, e.g., “Doing [3]”. To add a simple card count to a list without applying a limit, tag it #count.

This functionality is switched off by default.

Enable list limits and card counting

Default: off

Enable manual points on cards

When enabled, you can change how much a card is worth by adding a number in square brackets to the card text (you can also specify zero). Other extensions for Trello can sometimes prevent this option from working.

Default: off

Hide points in card text

If this box and “Points on cards” are checked, manual card points will be hidden.

⚠️ Note: this feature can interfere with other extensions for Trello that read or change card text, and vice versa. Disabling it should remove the conflict.

Default: off

Header cards

Sometimes it can be useful to split up long lists with header cards. Header cards are styled to look just like Trello list headers. To make a header card, start your card text with one or more hash symbols (# — make sure you leave a space after the hash); or two or more dashes (--); or forward slashes (//). Symbols at the beginning and end of the title will be stripped.

Enable header cards

Default: off

Add extra space above header cards

A little extra space can make header cards stand out better

Default: off

Separator cards

Separator cards create a blank gap between cards. You can use stickers on separator cards, but all other card content is hidden. To make a separator card, type two or more line symbols in a row, then no other text. Line symbols are dashes (--), equals signs (==) or underscores (__):

Enable separator cards

Default: off

Show separators as a visible line

Default: off
